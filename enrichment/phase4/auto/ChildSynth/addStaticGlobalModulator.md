Adds a static global modulator receiver to the specified modulator chain. Unlike `addGlobalModulator`, a static receiver provides a single modulation value per audio block rather than per voice. This is more efficient when voice independence is not needed - typical targets include effect parameters, FM depth, or synth-level controls. The `timeVariantMod` must be a reference to a modulator inside a GlobalModulatorContainer. Returns the new modulator as a Modulator handle, or undefined if creation fails.

> **Warning:** Some modulation targets only operate at block level. Using `addGlobalModulator` on these targets wastes CPU computing per-voice values that collapse to a single value. Default to `addStaticGlobalModulator` unless per-voice independence is specifically required (e.g. per-voice tremolo or vibrato).
