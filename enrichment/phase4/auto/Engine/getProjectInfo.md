Returns a JSON object containing project and company metadata from the project settings.

| Property | Description |
|----------|-------------|
| `Company` | Company name |
| `CompanyURL` | Company website URL |
| `CompanyCopyright` | Copyright string |
| `ProjectName` | Product/project name |
| `ProjectVersion` | Version string |
| `EncryptionKey` | Expansion encryption key |
| `HISEBuild` | HISE version used to build |
| `BuildDate` | Compilation date |
| `LicensedEmail` | Licensed user email (empty if copy protection not enabled) |
