Sets CSS pseudo-state selectors on this component for stylesheet-based styling. Multiple states can be combined in a single string (e.g. `":hover:active"`). Pass an empty string to clear all pseudo-states. The component automatically repaints after the state changes.

| Pseudo-state | Purpose |
|-------------|---------|
| `:hover` | Mouse hover |
| `:active` | Active/pressed |
| `:focus` | Keyboard focus |
| `:checked` | Toggled/checked |
| `:disabled` | Disabled |
| `:hidden` | Hidden |
| `:first-child` | First child |
| `:last-child` | Last child |
| `:root` | Root element |
