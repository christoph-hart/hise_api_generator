Registers a callback function for all table interactions. The callback receives a single argument - a JSON object with `Type`, `rowIndex`, `columnID`, and `value` properties. The `Type` property is one of these event type strings: Click, DoubleClick, Selection, ReturnKey, SpaceKey, DeleteRow, Slider, Button, ComboBox, SetValue. Must be called in onInit after `setTableMode()`.

> **Warning:** The `event.value` property changes meaning depending on `event.Type`: for Click/DoubleClick/Selection/ReturnKey/DeleteRow it is the full row data object; for Slider it is the numeric slider value; for Button it is the toggle state (boolean); for ComboBox it is the selected item. Always check `event.Type` before accessing `event.value`.
