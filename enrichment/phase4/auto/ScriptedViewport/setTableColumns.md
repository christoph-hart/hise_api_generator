Defines the columns of the table. Pass an array of column definition objects with at least an `ID` property. Each object can include `Type` (Text, Button, Slider, ComboBox, Image, Hidden), `Label`, `Width`, `MinWidth`, `MaxWidth`, `Visible`, and `Focus`. Additional properties depend on the cell type - Button columns use `Toggle` and `Text`, Slider columns use `MinValue`/`MaxValue`/`SkewFactor`/`StepSize`, and ComboBox columns use `items`, `ValueMode`, and `Value`. Must be called in onInit after `setTableMode()`.

> **Warning:** Set `"Focus": false` on Button columns that should not participate in arrow-key navigation. Without this, the focus order includes every column, making keyboard navigation slow when there are many action buttons.
