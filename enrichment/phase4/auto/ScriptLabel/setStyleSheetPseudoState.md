Sets CSS pseudo-state selectors for this component and triggers a repaint. Pass an empty string to clear all pseudo-states.

| Value | Description |
|---|---|
| `":first-child"` | First child state. |
| `":last-child"` | Last child state. |
| `":root"` | Root element state. |
| `":hover"` | Mouse hover state. |
| `":active"` | Active or pressed state. |
| `":focus"` | Keyboard focus state. |
| `":disabled"` | Disabled state. |
| `":hidden"` | Hidden state. |
| `":checked"` | Checked or toggled state. |
