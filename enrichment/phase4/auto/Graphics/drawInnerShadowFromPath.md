Draws an inner shadow inside the outline of a Path object, creating an inset/recessed visual effect. Unlike `drawDropShadowFromPath` which renders the shadow extending outward, this method renders the shadow extending inward. The `offset` parameter shifts the apparent light direction - positive x shifts the shadow rightward (light from left), positive y shifts it downward (light from above).

If the path is empty or invalid, the method silently does nothing.
