Registers a callback that fires when the DAW host begins or ends a parameter gesture (touch automation). The callback receives three arguments: the parameter type (0=Macro, 1=CustomAutomation, 2=ScriptControl, 3=NKS), the slot index within that type, and a boolean indicating whether the gesture is starting. Use this to provide visual feedback during automation recording - for example, highlighting a control while the DAW is touching it.
