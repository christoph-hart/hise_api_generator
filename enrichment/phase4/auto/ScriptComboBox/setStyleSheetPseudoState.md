Sets CSS pseudo-state selectors on this component for stylesheet-based styling. Multiple states can be combined in one string (e.g. `":hover:active"`). Pass an empty string to clear all pseudo-states. Automatically triggers a repaint.

Supported pseudo-states: `:first-child`, `:last-child`, `:root`, `:hover`, `:active`, `:focus`, `:disabled`, `:hidden`, `:checked`.
