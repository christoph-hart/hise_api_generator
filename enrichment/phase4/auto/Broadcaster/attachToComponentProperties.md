Registers the broadcaster as a source that fires whenever one of the specified properties changes on any of the specified components. Pass component names and property names as single values or arrays. Every component in the list must support every specified property -- for example, you cannot include a `ScriptPanel` in a list monitoring `viewPositionX` (a viewport-only property). If any property is invalid for any component, the entire attachment fails.

> [!Warning:$WARNING_TO_BE_REPLACED$] This method does not dispatch initial property values to existing targets on attachment. If listeners are added before this source, they receive no callback until the next actual property change. Attach the source before adding listeners for consistent initialisation, or call `resendLastMessage` afterwards.
