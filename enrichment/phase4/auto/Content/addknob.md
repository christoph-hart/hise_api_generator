Creates a ScriptSlider component (displayed as a rotary knob by default, changeable via the `mode` property) and adds it to the interface. Despite the name `addKnob`, the returned object is a ScriptSlider that supports both rotary and linear slider modes. Accepts 1 argument (name only, position defaults to 0,0) or 3 arguments (name, x, y). The call is idempotent - if a component with that name already exists, the existing component is returned. Whether the position is updated for existing components depends on `Content.setUpdateExistingPosition()`.
