Creates a ScriptPanel component - the most versatile component type, supporting custom paint routines, mouse and key event callbacks, animation timers, drag-and-drop, and child component parenting. Accepts 1 argument (name only, position defaults to 0,0) or 3 arguments (name, x, y). The call is idempotent - if a component with that name already exists, the existing component is returned. Whether the position is updated for existing components depends on `Content.setUpdateExistingPosition()`.
