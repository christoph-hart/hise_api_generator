Returns a string representation of the file path in the specified format. Pass one of the format constants available on any File object: `FullPath` (0) for the complete absolute path, `NoExtension` (1) for the filename without extension, `Extension` (2) for just the file extension including the dot, or `Filename` (3) for the filename with extension. Use `NoExtension` for display names and `FullPath` for constructing lookup keys.
