Writes the given text string to this file, replacing any existing content. The write is atomic - it uses a temporary file to prevent data loss if the write fails. Returns `true` on success. This is the low-level text writing method that `writeObject` and `writeAsXmlFile` delegate to internally. A common use case is writing HISE link files (`LinkWindows` / `LinkOSX`) containing an absolute path to redirect the sample folder location.
