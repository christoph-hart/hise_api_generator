MidiProcessor::restoreScriptControls(String base64Controls) -> undefined

Thread safety: UNSAFE -- base64 decoding, ValueTree parsing, and state restoration involve heap allocations.
Restores UI control values from a base64-encoded string previously generated by
exportScriptControls(). Only works on script processors (JavascriptMidiProcessor).
Restores only scripting content (knob/slider/button values) without triggering
script recompilation.
Pair with:
  exportScriptControls -- generate the base64 string
  restoreState -- full state alternative that works on all module types (triggers recompile on script processors)
Anti-patterns:
  - Do NOT call on built-in MIDI modules -- throws "restoreScriptControls can only be used on Script Processors". Use restoreState() instead.
