# Video Tutorials

---
title: Video Tutorials
description: A searchable and cross-referenced inventory of David Healeys video tutorials
---

This section contains compressed transcripts from video tutorials by [David Healey](https://www.youtube.com/@DavidHealey), which is an excellent source of knowledge for working in HISE.

Each video has been processed into a concise, actionable recipe that preserves the key knowledge, best practices, and code examples. These recipes are not a replacement for the original videos - they are a searchable, skimmable reference and used for cross-referencing across the documentation to increase the visibility of that knowledge. Where applicable, code blocks are provided as copy-pasteable HiseScript. Each section links back to the exact timestamp in the original video via YouTube deep links.

If you find these resources helpful, consider supporting David's work on [Patreon](https://www.patreon.com/davidhealey) or subscribing to his [YouTube channel](https://www.youtube.com/@DavidHealey). Also smash that like button.

> In additions to these tutorials, there is also the excellent and free [HISE Bootcamp](https://audiodevschool.com/courses/hise-bootcamp/) - these videos are not contained in this section, just enroll in this course and chew your way through it.
